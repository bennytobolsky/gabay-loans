# מרכז הלוואות חכם — גבאי

צ'אט AI מבוסס Claude שעונה על שאלות לגבי מסלולי הלוואות פנסיוניות, מוכן להעלאה לענן.

## מה יש בתיקייה

- `index.html` — האתר עצמו (חזית, סטטי).
- `api/chat.js` — שרת קטן (Serverless Function) שמחזיק את מפתח ה-API של Anthropic **בצד השרת בלבד** ומעביר אליו את הבקשות מהצ'אט. זה הכרחי: לא ניתן לשים מפתח API בקוד של הדפדפן, כי כל מבקר באתר יוכל לגנוב אותו ולהשתמש בו על חשבונך.
- `.env.example` — תזכורת לשם משתנה הסביבה שצריך להגדיר (`ANTHROPIC_API_KEY`). לא מכיל מפתח אמיתי.
- `קובץ מרכז הלווואת.xlsx` — קובץ המקור של נתוני ההלוואות (הנתונים כבר משובצים בתוך `index.html`, הקובץ נשמר לעיון/עדכון עתידי).

## שלב 1 — קבלת מפתח API מ-Anthropic

1. היכנס ל-https://console.anthropic.com
2. צור חשבון (אם אין לך) והגדר אמצעי תשלום — השימוש בצ'אט מחייב תשלום לפי צריכה.
3. בתפריט "API Keys" צור מפתח חדש והעתק אותו (הוא יוצג פעם אחת בלבד).

## שלב 2 — העלאה לענן עם Vercel (הכי פשוט)

Vercel הוא שירות אירוח בחינם (לשימוש כזה) שמריץ גם קבצים סטטיים וגם פונקציות שרת קטנות כמו `api/chat.js`, בלי צורך בשרת משלך.

### אפשרות א׳: דרך האתר (בלי טרמינל)

1. העלה את התיקייה הזו ל-GitHub (צור ריפו חדש בשם למשל `gabay-loans` ב-https://github.com/new והעלה אליו את כל הקבצים — אפשר גם לגרור קבצים דרך ממשק ה-Web של GitHub).
2. היכנס ל-https://vercel.com והתחבר עם GitHub.
3. לחץ "Add New" → "Project", בחר את הריפו `gabay-loans`.
4. במסך ה-Import, תחת "Environment Variables" הוסף:
   - Name: `ANTHROPIC_API_KEY`
   - Value: המפתח שקיבלת משלב 1
5. לחץ "Deploy". תוך דקה תקבל כתובת אינטרנט חיה כמו `gabay-loans.vercel.app`.
6. את הכתובת הזו אפשר לשלוח לכל מי שצריך להשתמש במערכת — היא תעבוד לכל מי שיש לו את הקישור, בלי צורך בהתקנה.

### אפשרות ב׳: דרך שורת הפקודה (מהר יותר אם נוח לך עם טרמינל)

```
npm install -g vercel
cd "C:\Users\bennyt\Desktop\gabay-loans"
vercel login
vercel
```

בפעם הראשונה תתבקש כמה שאלות (project name וכו') — אפשר לאשר את ברירות המחדל. בסיום תקבל קישור preview.

אחר כך הגדר את מפתח ה-API (פעם אחת):
```
vercel env add ANTHROPIC_API_KEY production
```
(יבקש שתדביק את המפתח).

ולבסוף עלייה לסביבת הפרודקשן (הכתובת הסופית והקבועה):
```
vercel --prod
```

## עדכונים עתידיים

כל שינוי שתעשה ב-`index.html` או ב-`api/chat.js`:
- אם עבדת דרך GitHub: פשוט תעלה (`git push`) את השינוי — Vercel יבנה ויפרסם גרסה חדשה אוטומטית.
- אם עבדת דרך ה-CLI: הרץ שוב `vercel --prod`.

## הערות חשובות

- **עלות**: הקישור פתוח לכל מי שמקבל אותו, וכל שאלה בצ'אט צורכת קרדיט מהחשבון שלך ב-Anthropic. אם תרצה בעתיד להגביל גישה (סיסמה) או להוסיף הגבלת קצב שימוש, זה ניתן להוספה בקלות ב-`api/chat.js`.
- **הנתונים**: כרגע נתוני ההלוואות קבועים בתוך `index.html` (המשתנה `DATA`). כדי לעדכן ריביות/אחוזים יש לערוך את המשתנה הזה ולפרסם מחדש.
